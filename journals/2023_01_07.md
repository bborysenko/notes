- ## [[How to add icon to a Logseq page?]]
	- You can use `icon::` [built-in property](https://docs.logseq.com/#/page/built-in%20properties) to define icon identifier to a page so that it shows it next to page name under "Favorites" & "Recent" sections. Right now it only supports emojis, but there is a [feature request for asset support](https://discuss.logseq.com/t/image-asset-as-page-icon/3826). If you want to see it elsewhere in your note (when you refer it), you'll probably want to look into [logseq-reference-styles](https://github.com/benjaffe/logseq-reference-styles) plugin.
- ## [[How to upload books to Kindle?]]
	- You can use Send to Kindle free service to send documents to your Kindle library. Upload documents through the [web](https://www.amazon.com/sendtokindle), an [authorized email address](https://www.amazon.com/sendtokindle/email), the Kindle app for [iOS](https://www.amazon.com/sendtokindle/ios) and [Android](https://www.amazon.com/sendtokindle/android) devices, or a Send to Kindle application if available. All documents sent through Send to Kindle are protected with end to end encryption.
	- Supported file types: EPUB, PDF, TXT, RTF, DOC, DOCX, HTM, HTML, PNG, GIF, JPG, JPEG, BMP. Max file size: 200 MB.
- ## [[How to export highlights from Kindle?]]
	- You can export highlights using and Kindle Desktop and then format to Markdown with [kindle-formatter.com](https://kindle-formatter.com/).
